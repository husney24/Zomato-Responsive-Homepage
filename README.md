## How to Run

1. Clone the repo:
   ```bash
   git clone https://github.com/husney24/Zomato-Responsive-Homepage.git
   ```

2. Go to the Zomato folder:
   ```bash
   cd Zomato-Responsive-Homepage/Zomato
   ```

3. Open `index.html` in your browser or use Live Server

That's it!
